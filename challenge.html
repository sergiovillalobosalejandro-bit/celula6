Bienvenidos al Challenge HTML

Este proyecto es un ejercicio práctico para aprender a trabajar colaborativamente usando Git y GitHub. Cada miembro del equipo contribuye con una parte específica del código.

Sobre el Proyecto

El es construir una página web completa donde cada integrante aporta diferentes elementos: estructura, header, contenido y footer.

Nuestro Equipo

Somos un equipo de 4 desarrolladores excelentemente buenos, y estamos aprendiendo las mejores prácticas de colaboración con control de versiones.
